## 🌍 English Description

### 🎮 Overview  
**Chemistry Challenge** is a fully custom-built interactive web game created entirely from scratch by **Haitham Muneer Al-Jabri**.  
It transforms traditional chemistry lessons into a dynamic, competitive, and visually engaging experience that combines education and entertainment.  

Inspired by *Jeopardy!*, this project reimagines the concept of classroom quizzes through a chemistry-based, two-team duel system with lifelines, animations, and elegant transitions.  

---

### 🧠 Development Journey  

**1. Concept & Brainstorming**  
The idea began with the vision to make chemistry lessons more exciting through gamification — turning study sessions into a lively competition instead of static Q&A.  

**2. Design & Planning**  
All features were planned in detail: timers, scoring logic, lifelines, animations, and the overall flow from start to victory.  
User experience and animation smoothness were top priorities.  

**3. Visual Identity**  
The interface was designed manually to reflect a scientific aesthetic — with animated beakers, molecules, and soft gradients inspired by lab environments.  

**4. Manual Development**  
Built completely from scratch using **HTML, CSS, and JavaScript**, later integrated with modern technologies like **React, TypeScript, TailwindCSS, Vite**, and **shadcn-ui** for refined UI.  
No templates or auto-generated layouts were used.  
Every logic, interaction, and animation was written manually by the developer.  

**5. Smart Use of AI Tools**  
AI was used selectively for code review, color palette tuning, and design brainstorming — but all creativity, architecture, and logic were handcrafted.  
This project is proof that AI can enhance human creativity, not replace it.  

**6. Testing & Polishing**  
Several iterations were tested to ensure perfect timing, responsive layout, and bug-free performance on various screen sizes.  

---

### 💡 Features  

- 🎨 Smooth, high-quality animations and transitions.  
- ⚔️ Two-team competitive gameplay with custom team names.  
- 🧩 12 total questions (4×200, 4×400, 4×600).  
- 🧠 Three lifelines:
  - Multiple-choice assist  
  - Teacher-help timer  
  - Double-points (risk-based reward system)  
- ⏱️ Dual-timer system for team turns and responses.  
- 🧪 Chemistry-themed visual interface with color-coded teams.  
- 🎵 Interactive sound effects and celebration animations.  
- 🧭 Full manual control for timing, scoring, and question states.  
- 💻 Fully responsive and optimized for classroom use.
